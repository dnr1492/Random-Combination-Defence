●BM 구조

**보석 결제 -> 캐릭터 또는 유물 뽑기

*광고 보기
-> 1회 시청할 시 일정량의 보석, 골드 획득
=> 광고 제거 + 2배속

*캐릭터 뽑기
-> 기본 레전드 캐릭 외 레전드 캐릭은 뽑기로만 획득
-> 보유하고 있는 캐릭을 중복으로 뽑을 시 해당 캐릭의 카드를 1장씩 획득
-> 일정량의 해당 캐릭의 카드로 레벨업
-> 레벨 증가에 따른 공격력, 공격 속도, 스킬 계수 등 증가
-> 특정 레벨 도달 시 스킬 잠금 해제 등의 효과 부여

*유물 뽑기 ('운빨존많겜 유물 시스템 참고')
=> 전체적인 플레이어의 데미지 증가
=> 전체적인 플레이어의 공속 증가
=> 전체적인 플레이어의 이동 속도 증가
=> 전체적인 플레이어의 스킬 데미지 증가
=> 전체적인 플레이어의 스킬 발동 확률 증가
=> 보스에게 가하는 데미지 증가
=> 몬스터 처치 시 골드 획득량 증가
=> 시작 골드 보유량 증가
=> 웨이브 지급 골드량 증가
=> 몬스터 카운트 증가
=> 플레이어 판매 성공 확률 증가
=> 모든 적의 이동 속도 둔화
=> 모든 적의 방어력 저하
