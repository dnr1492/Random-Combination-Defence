*안흔한 (13)
Melee : 기사 검투사 암살자 순찰자 사냥꾼 싸움꾼 버서커 파이크맨

*조합식
전사 + 전사 = 기사
전사 + 격투가 = 검투사
도적 + 도적 = 암살자
도적 + 궁수 = 순찰자
궁수 + 궁수 = 사냥꾼
격투가 + 광전사 = 싸움꾼
광전사 + 광전사 = 버서커
창술사 + 창술사 = 파이크맨(다중타격)
창술사 + 마법사 = 마창사(근접 마법)
주술사 + 주술사 = 흑마술사(디버프)
주술사 + 사제 = 신비술사(상태이상)
마법사 + 마법사 = 대마법사
사제 + 사제 = 성직자(버프)
